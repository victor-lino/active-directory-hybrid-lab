# Troubleshooting / Problemas e Soluções

## Common Issues / Problemas Comuns
1. DNS not resolving AD names
   - Fixed by setting AD server as DNS for clients
2. User login fails
   - Checked OU placement and replication
3. Access to SMB shares denied
   - Verified NTFS + SMB permissions
4. Azure Connect sync errors
   - Re-run Delta Sync and check Entra provisioning logs
